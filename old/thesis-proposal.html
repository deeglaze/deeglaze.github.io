Thesis for James Ian Johnson

  1. Title: Systematic constructions for higher-order program analyses
  2. Abstract:

    Program analyses are crucial for enabling compiler optimizations, program verification and conversely bug-finding. However, analysis tools are difficult to design, engineer and use. In this proposal, I describe a collection of systematic semantics refactorings that transform an original programming language semantics into sound, computable, precise and efficient program analyses. I additionally propose a new semantics for temporal higher-order contracts in order to statically verify temporal properties of higher-order programs within the previously described framework.
